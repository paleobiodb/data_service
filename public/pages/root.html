About

Thanks to our membership, which includes 350 scientists from 133 institutions in 24 countries, the Paleobiology Database is able to provide scientists and the public with information about the fossil record. The database is currently undergoing a major renovation to better serve its users. Stay tuned! In the meantime, check out the FAQ and the people who make this database possible.

API

The Paleobiology Database Application Programming Interface (API) gives scientists, students, and developers programmatic access to taxonomic, spatial, and temporal data contained within the database. To get started building your own application for data analysis, visualization, or acquisition, head over to the API documentation.

Apps

Built on the Paleobiology Database API, there are many applications that put our data to use. Can't find an application that fits your needs or matches your vision? Check out the API documentation and learn how to build your own application. When you are ready to share your work, submit your application to our app gallery so that we can help spread the word.

Friends, visitors, and members of the Paleobiology Database:

Welcome to the new Paleobiology Database!

We have a new temporary look and feel to the site. We’ll be trying out several new designs in the near future, so don’t be surprised if you see a new design the next time you visit.

Besides a new look, some more significant changes will soon arrive to the PaleoDB. Our IT team is developing numerous exciting new features, including an API that will empower you to develop websites, mobile applications, desktop applications, and analytical scripts that directly access the rich public data within the PaleoDB. Check back in the API section for updates about our progress, and feel free to look at some applications in the Apps section.

We are all excited about the new directions PaleoDB is heading. We aim to be the premier place for hosting and obtaining paleobiological data that are of use to paleontologists, students, and the world. Please give us your feedback so that we can learn more about how you use our data, and how to make our service better fit your needs.
