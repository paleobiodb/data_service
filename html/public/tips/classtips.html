Commonly Used Class Names in the Paleobiology Database

This is a partial list of the valid class names commonly used in the database. The list is in turn based on Sepkoski's unpublished genus-level classification, which is used for the higher-order taxonomy lookups done by the collection search engine. If you know of any missing names, please write to tell me about them.
