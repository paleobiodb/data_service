Tip Sheet for the Reference Search and Entry/Editing Forms

This page was written by Arnie Miller and John Alroy in March 1999 and is maintained by Alroy.

searching for references - entering and editing references

Searching for references

Data contributor
This is a required field. If your name is not on the list, chose the name of the person you are working with.

First author's last name
This is a required field. It and the next field are needed to perform a search of the database's existing list of references.

Publication year
This is an optional field, but it can be helpful if you want to avoid getting back too many matches to your search.

Entering and editing references

"Choose one" list of references
This is a required choice. WARNING: if you find a match, it is possible that someone else already has entered the collection.

First author's initials
This is a required field, except when entering institutions or anonymous authors. Do not spell out the author's full name. Do put a period after each initial, and a space between each pair of initials. One, two, or three initials are OK.

First author's last name
This is a required field. The value anon. or anonymous can be used for unknown author(s). Hyphenated names are OK, as are spaces separating parts of multi-word names like "van Rijn."

Second author's initials
Not required for single-author papers. Third, fourth, etc. authors are not listed here, but in the comment field below.

Second author's last name
Not required.

Initials and names of additional authors
List the the third, fourth, fifth, etc. authors in order, initials first. Comma separators are optional. The word "and" before the last author is optional and not recommended.

Publication year
This is a required field. Use the actual date of publication, if available, not the nominal year of publication for the volume.

Publication type
Choose the closest possible value. "unpublished" includes field notes, manuscripts, undergraduate theses, etc.; "guidebook" includes individual articles in guidebooks.

Title
This is a required field. Type in the entire title and do not use abbreviations. Use of colons and semi-colons is OK. If the title is not in English, enter the original title first, and then enter an English-language translation in brackets. Example:

Étude sur les rostroconches du Danien supérieur de Topeka [A study of the upper Danian rostroconchs of Topeka]
For a thesis, enter only the name of the institution (e.g., "University of Saskatchewan").

Full name of serial
This is a required field. If you are citing a book, enter the full name here, following this style:
B. Dobbs and B. Young, eds., Drab Gray Fossils From Lithuania. Hokum Press, Lagos, Nigera.
If you are citing a book chapter, enter the same information but start the entry with in (as in "in B. Dobbs and B. Young, eds., ...").

Volume
Required only for serials, not for books.

Number
Optional.

First page
This is a required field. Do NOT add a hyphen after the page number; this will be done automatically by the data entry processing program. If you are citing an entire book, put "1" here and the total number of pages in the last page field. If the pages are numbered with combinations of numbers and letters (e.g., "A-32") put everything in an do not strip out the letters.

Last page
Required only if the paper actually is longer than one page (many abstracts have only one page).
