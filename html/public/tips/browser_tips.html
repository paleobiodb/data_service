Tip Sheet for Web Browsers

Microsoft Internet Explorer is not recommended for use on our web site at this time due to issues with session management, Cascading Style Sheets (CSS), and Java Script. We instead recommend that you use one of the more standards compliant browsers listed below. Click on the links to go to a download page for each browser. The listing includes free browsers for each common operating system.

Windows

  • Mozilla (free)
  • Mozilla Firefox (free)

Mac OS X

  • Safari (free)
  • Omni Web (free trial)
  • Mozilla (free)
  • Mozilla Firefox (free)
  • Camino (free)

Mac OS 9 or earlier

  • Mozilla 1.2.1 (free)
  • Netscape 7 (free)

Linux

  • Mozilla (free)
  • Mozilla Firefox (free)
  • Konqueror (free)
Note: the web browser Opera (free trial) should work as well, although we have not tested it.
