Tip Sheet for Web Browsers

Microsoft Internet Explorer is not recommended for use on our web site because it is not compliant with universal HTML, Cascading Style Sheet (CSS), and JavaScript standards. In other words, it just doesn't work right in all sorts of ways, which is why programmers around the world hate it passionately. We instead recommend that you use one of the more standards compliant browsers listed below. Click on the links to go to a download page for each browser. The listing includes free browsers for each common operating system.

Note: we are not asking you to replace Internet Explorer, but simply to download one of these browsers in addition to it. Feel free to use IE for your other browsing needs if you still prefer it.

We recommend Mozilla Firefox, which will work either on Windows or Mac OS X, or Safari, which is the official Mac OS X browser.

You may also want to read the Mozilla Firefox FAQ which describes some other general reasons why people should not use Internet Explorer.
