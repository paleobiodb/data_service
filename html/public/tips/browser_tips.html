Tip Sheet for Web Browsers

Microsoft Internet Explorer is not recommended for use on our web site at this time due to issues with session management, Cascading Style Sheets (CSS), and Java Script. We instead recommend that you use one of the more standards compliant browsers listed below. Click on the links below to go to a download page for each browser. The listing includes free browsers for each common operating system.

Note: we are not asking you to replace Internet Explorer, but simply to download one of these browsers in addition to it. Feel free to use IE for your other browsing needs if you still prefer it.

If you are not familiar with browsers other than Internet Explorer, here is a brief description. Mozilla is a combination of a web browser, email/news client, and web design program. Mozilla Firefox is a web browser only and has a more modern user interface than the regular Mozilla. Safari is a web browser which only works on Mac OS X. We typically use Firefox, Safari, and Mozilla (in that order) for development and testing of the PBDB web site.

You may also want to read the Mozilla Firefox FAQ which describes some other general reasons why people should not use Internet Explorer.
