Occurrence table tip sheet


  • The abundance unit of all occurrences will be changed to reflect the value selected in the "Abund. unit" pulldown. Thus, all occurrences on the tip sheet with abundance data must have the same abundance unit.

  • If no abundance data are given, use "x" to denote a taxon's presense.

  • Abundances colored red can't be edited because they belong to a different authorizer.

  • Abundances denoted "category" or "rank" may be autofilled if you hit the TAB key, if similar values already exist in the form. For example, if you fill in "Equus sp." as "rare" for the first entry, you can enter "r" and hit tab for the second entry to autofill to "rare."

  • Please see the collection and occurrence data tip sheet as well.
