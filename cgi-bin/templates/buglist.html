Names of Database members who have reported the bugs are given in italics. Please also see our list of new pages, scripts, and other features.

Krug) - Apparently fixed. Could someone confirm this for me?
  • Users are logged out automatically whenever they go from contributors-only areas to the public web site, so getting back requires another login (J. Alroy)

Cosmetic bugs

  • Possibly, stray collection ID numbers in some versions of the reference search return page (J. Alroy)

Data errors

  • Several dozen records include the word "Series" in the epoch fields or "Stage" in the local age/stage field; these should be stripped (JA)
  • Quotation marks have been entered into some of the taxon name fields (JA)

Feature requests

NOTE: all requests on this list will be implemented if no objections are raised by Database members.
  • Diversity curves restricted by paleoenvironment and not just by environmental zone (W. Kiessling, 15 December 2003)
  • Downloads restricted by collection's creation date (W. Kiessling, 10 November 2003)
  • Collection searches return number of occurrences in each collection (M. Foote, 17 October 2002)
  • Genus names checked for classification data on entry using occurrences form; shortcut to enter taxonomy data quickly for new names (W. Kiessling, 22 August 2002)
  • Curve, download, map, and report modules should query reIDs and not just occurrences when query parameters include genus name (J. Alroy, 16 August 2002)
  • Options for sorting collections matched by a search (e.g., by geological age) (Ross Secord, 16 August 2002)
  • New stratigraphic unit, stratigraphic section, and geochronological data tables keyed to each other and to the collection table (Advisory Board, 11 August 2002)
  • Search of references by research group, exploiting the fact that group membership is defined in the MySQL person table (H. Sims, 27 July 2002)
  • "animal" value in plant organ pulldowns (?) (H. Sims)
  • On the collection search form, genus name searches do not allow partial string matching (M. Foote)
  • On the collection search form, searches by continent, taxon ID comment, lithology and geology comments, and author should be allowed (M. Foote)
  • The list of original biominerals on main form should be revised: "phosphate" will be changed to "hydroxyapatite"; several other values will be added (taphonomy group)
  • The third of three taphonomy tip sheets needs to be formatted properly (taphonomy group)
  • Separate collection size field into two: one with MNI and one with number of specimens. If both are known then both can be recorded, if only one is known then only one needs to be recorded (VP group)
  • Geochronological age estimates need to be tracked in a table/form documenting estimate, error, method, stratigraphic position and/or foreign key to collection(s), and comments (paleobotany/VP groups)
  • Tertiary lithology fields (W. Kiessling)
  • Carbonate geometry field (values need to be discussed) (W. Kiessling)
  • "replaced by chlorite" checkbox (W. Kiessling)
  • Occurrence matrix data ingest tool (paleobotany group)
  • Plant organ tip sheet needs to be updated (paleobotany group)
