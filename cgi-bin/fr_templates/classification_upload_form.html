Instructions:
  • Please "Save as" in Excel 95,Excel 97, or Excel 2000 format is the file is created with a newer version of Office.
  • The first line of the excel file must be a header. The first column in the header must be the rank of the parent (taxon being assigned to), the second column must be the the rank of the child (taxon being classified). Valid ranks are: order, class, family, genus.
  • The first 11 columns must be in the following format: [higher order taxon, lower order taxon, author, pub. year, first appearance, last appearance, phylogenetic status, habitat, first occurrence, last occurrence, comments]. See description of columns.
  • First appearance and last can be time intervals from any scales entered in the system. See list of time scales.
  • Previous uploads may be resumed by coming back to this page.
