Data entered this month

415 taxonomic names, 1108 classification and synonymy opinions, 214 references, 555 collections, and 4265 occurrences

New script and database structure features

  • Data now provided to GEON's Paleo Integration Project using web services
  • Faster lookups of period, epoch, age/stage, and bin names
  • Multiple changes to handle opinions that taxon names are misspelled (lapsus calami records)
