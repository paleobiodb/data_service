Meetings

  • Taphonomy working group, NCEAS, 20 - 23 September
  • Taxonomy working group, Smithsonian Institution, 10 - 12 September

New script and database structure features

  • New script for entry and editing of ecological and taphonomic data for taxa
  • New script for printing hierarchical taxonomic classifications to the screen and to a file
  • New function in the taxonomic information script for printing synonymy lists in the traditional format
  • New function in the taxonomic information script for using author names and/or publication years to search for taxa
  • New function in the download script for downloading lists of genera instead of collections or occurrences
